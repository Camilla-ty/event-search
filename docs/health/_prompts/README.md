# Health Check execution prompts

Canonical, paste-ready execution prompts for Engineering Health Check reviews.

| Review | Cadence | Prefix | Prompt |
|---|---|---|---|
| Code Hygiene Audit | Monthly | `HYG` | [`code-hygiene.md`](./code-hygiene.md) |
| Product Audit | Monthly | `PROD` | [`product.md`](./product.md) |

Prompts inherit Framework **v1.1** rules from [`../README.md`](../README.md) and ownership from [`../audit-catalog.md`](../audit-catalog.md). Reports use [`../_templates/report-template.md`](../_templates/report-template.md).

**Note:** Architecture and Security execution prompts were designed earlier in chat but are not yet checked into this folder. Add them here when they are formalized. There is no legacy Dead Code prompt to preserve — Code Hygiene replaces that review type.
